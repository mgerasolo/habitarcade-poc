# Epic 7: Wallboard Mode

## Epic Overview
TV/large display optimized view for always-on dashboard.

## Priority
**Medium** - Week 2

## Stories

### Story 7.1: Wallboard Layout
**As a** user
**I want** a full-screen wallboard view
**So that** I can display on my TV

**Acceptance Criteria:**
- [ ] Full-screen mode (F11 or button)
- [ ] Hides browser chrome
- [ ] Optimized widget arrangement
- [ ] Shows Habit Matrix + Kanban + Time Blocks together
- [ ] URL route: /wallboard

---

### Story 7.2: Large Display Scaling
**As a** user
**I want** larger fonts/elements
**So that** I can read from across the room

**Acceptance Criteria:**
- [ ] Base font size increased (1.5x)
- [ ] Larger cell sizes (32px+)
- [ ] Increased spacing
- [ ] Readable from 10+ feet

---

### Story 7.3: Wallboard Typography
**As a** user
**I want** fonts optimized for distance
**So that** text is legible on TV

**Acceptance Criteria:**
- [ ] Increased font weights
- [ ] Higher contrast colors
- [ ] Category names prominent
- [ ] Date headers visible

---

### Story 7.4: Interactive Wallboard
**As a** user
**I want** to click cells on the wallboard
**So that** I can update habits from the TV

**Acceptance Criteria:**
- [ ] Status cycling works
- [ ] Larger click targets
- [ ] Visual feedback on interaction
- [ ] Auto-refresh every 30 seconds

---

### Story 7.5: Combined Widget View
**As a** user
**I want** key widgets visible together
**So that** I see everything at once

**Acceptance Criteria:**
- [ ] Habit Matrix (primary, large)
- [ ] Weekly Kanban (today + tomorrow visible)
- [ ] Time Blocks (current block prominent)
- [ ] Clock/date display
- [ ] No scrolling required

---

## Definition of Done
- Readable from 10 feet
- Interactive and responsive
- Works on 1080p TV
